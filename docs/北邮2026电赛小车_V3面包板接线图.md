# 北邮2026电赛小车 V3 面包板接线图

适用方案：`北邮2026电赛小车_简化电源架构与接线方案_V3.md`

本图按常见 `830 孔全尺寸面包板` 绘制：

- 中间插孔：`A-E` 为一组、`F-J` 为一组，同一列 5 个孔内部相通。
- 中间沟槽：`E` 和 `F` 不相通。
- 电源轨：上红轨为 `3V3`，上蓝轨和下蓝轨为 `GND`，下红轨为 `V5L`。
- `V5L` 来自 `TB6612#1 板载 5V OUT`，不是 LM2596。
- `V5S`、`VBAT`、电机输出线、舵机主电源线不上面包板。

---

## 1. 电源轨

```text
上红轨: 3V3  <- 5V->3.3V 小稳压模块 OUT+ / M0 3V3
上蓝轨: GND  <- GND_PWR / M0 GND
下红轨: V5L  <- TB6612#1 板载 5V OUT
下蓝轨: GND  <- GND_PWR
```

如果面包板电源轨中间断开，必须同名跨接：

```text
上红左段 3V3 -> 上红右段 3V3
上蓝左段 GND -> 上蓝右段 GND
下红左段 V5L -> 下红右段 V5L
下蓝左段 GND -> 下蓝右段 GND
上蓝 GND     -> 下蓝 GND
```

严禁：

```text
3V3 不直接接 V5L，只能通过 5V->3.3V 小稳压模块相连
V5L 不接 V5S
V5S 不上面包板长红轨
VBAT 不上面包板长红轨
电机输出线不上面包板
```

## 1A. 从电池开始的完整供电接线

先只看供电，不接信号线。

```text
2S 锂电池 XT30 +
  -> 10A 保险丝
  -> 船型总开关
  -> 急停常闭触点
  -> VBAT 分配点
       -> 左侧 TB6612#1 VIN/VM/电源+
       -> 右侧 TB6612#2 VIN/VM/电源+

2S 锂电池 XT30 -
  -> GND_PWR 分配点
       -> 左侧 TB6612#1 GND/电源-
       -> 右侧 TB6612#2 GND/电源-
       -> M0 GND
       -> 面包板 GND 轨
```

M0 的 3.3V 这样给：

```text
左侧 TB6612#1 板载 5V OUT
  -> V5L 星形点
       -> 5V->3.3V 小稳压模块 IN+
       -> 面包板下红轨 V5L
       -> 灰度模块 5V

5V->3.3V 小稳压模块 OUT+
  -> M0 3V3
  -> 面包板上红轨 3V3

M0 GND
  -> GND_PWR 分配点
  -> 面包板 GND 轨
```

3V3 逻辑电源这样分配：

```text
5V->3.3V 小稳压模块 OUT+
  -> 面包板上红轨 3V3
       -> M0 3V3
       -> TB6612 逻辑 VCC
       -> IMU/OLED/蜂鸣器/编码器参考电源，按模块需要接
```

右侧 TB6612#2 的 5V 输出不要给 M0：

```text
右侧 TB6612#2 板载 5V OUT
  -> V5S 星形点
       -> PCA9685 V+ / SG90 预留
       -> KY-008 激光预留
```

上电前必须先测：

```text
左侧 TB6612#1 5V OUT-GND = 5.0V 左右
右侧 TB6612#2 5V OUT-GND = 5.0V 左右
V5L 与 V5S 正极不导通
V5L 与 3V3 不直接导通；若已接稳压模块，只能看到经模块输入/输出形成的非短路路径
VBAT 不接到面包板红轨
```

USB 调试注意：

```text
第一次烧录/调试可以只插 M0 USB，不接外部 3V3。
整车电池供电测试时，再接 V5L -> 5V->3.3V 小稳压模块 -> M0 3V3。
如果 M0 同时插 USB 和外部 3V3，先确认 LaunchPad 不会 3.3V 回灌；不确定时只用一种主控供电。
```

## 3. 各列接线表

### 3.1 TB6612 逻辑并线区 C01-C07

`A/B/C/D/E` 同一列内部相通，所以同一个网络直接插同一列。

| 列 | 网络 | 插线方式 |
| ---: | --- | --- |
| `C01` | `PWM_L` | M0 `PA12`、左侧 TB6612#1 `PWMA`、左侧 TB6612#1 `PWMB` 都插本列 |
| `C02` | `PWM_R` | M0 `PA13`、右侧 TB6612#2 `PWMA`、右侧 TB6612#2 `PWMB` 都插本列 |
| `C03` | `L_IN1` | M0 `PA8`、左侧 TB6612#1 `AIN1`、左侧 TB6612#1 `BIN1` 都插本列 |
| `C04` | `L_IN2` | M0 `PA27`、左侧 TB6612#1 `AIN2`、左侧 TB6612#1 `BIN2` 都插本列 |
| `C05` | `R_IN1` | M0 `PB0`、右侧 TB6612#2 `AIN1`、右侧 TB6612#2 `BIN1` 都插本列 |
| `C06` | `R_IN2` | M0 `PB6`、右侧 TB6612#2 `AIN2`、右侧 TB6612#2 `BIN2` 都插本列 |
| `C07` | `STBY` | M0 `PB18`、两块 TB6612 `STBY`、10k 上拉电阻下端都插本列 |

电机输出对应：

```text
左侧 TB6612#1:
  AOUT -> LF 左前
  BOUT -> LR 左后

右侧 TB6612#2:
  AOUT -> RF 右前
  BOUT -> RR 右后
```

STBY 上拉：

```text
R3 10k:
  一端 -> 上红轨 3V3
  另一端 -> C07 任意 A-E 孔
```

TB6612 逻辑电源：

```text
两块 TB6612 VCC -> 上红轨 3V3
两块 TB6612 GND -> GND 轨，同时回 GND_PWR
两块 TB6612 VM/VIN -> VBAT，外部粗线，不进面包板
两块 TB6612 5V OUT -> #1 到 V5L，#2 到 V5S，二者不并联
两块 TB6612 3V3 OUT -> 默认悬空
```

### 3.2 灰度模块区 C10-C15

| 列 | 网络 | 接法 |
| ---: | --- | --- |
| `C10` | `V5L` | 跳到下红轨 `V5L`，灰度 `5V` 插本列 |
| `C11` | `GND` | 跳到 GND 轨，灰度 `GND` 插本列 |
| `C12` | `GRAY_AD0` | M0 `PB7`、灰度 `AD0` 插本列 |
| `C13` | `GRAY_AD1` | M0 `PB8`、灰度 `AD1` 插本列 |
| `C14` | `GRAY_AD2` | M0 `PA22`、灰度 `AD2` 插本列 |
| `C15` | `GRAY_OUT` | M0 `PB4`、灰度 `OUT` 插本列 |

灰度局部去耦：

```text
C1 0.1uF:
  一端 -> C10 / V5L
  另一端 -> C11 / GND
```

### 3.3 编码器区 C18-C26

如果你的 MG310 是整束 6P/多芯线直接插到轮趣 TB6612 驱动板，那么这里的“编码器区”不是接电机本体，而是接驱动板上对应的编码器输出/排针。电机所有线仍先接 TB6612；M0 第一版只从左后 `LR` 和右后 `RR` 拿 `A/B` 脉冲。

| 列 | 网络 | 接法 |
| ---: | --- | --- |
| `C18` | `ENC_VCC_REF` | 若左侧 TB6612#1 编码器排针要求外部逻辑电源，可接 `3V3`；若驱动板已给编码器供电，本列留空 |
| `C19` | `GND` | 左侧 TB6612#1 的 `LR` 编码器接口 `GND`，跳到 GND 轨 |
| `C20` | `LR_A` | M0 `PB19`、左侧 TB6612#1 的 `LR` 编码器输出 `A` |
| `C21` | `LR_B` | M0 `PB24`、左侧 TB6612#1 的 `LR` 编码器输出 `B` |
| `C23` | `ENC_VCC_REF` | 若右侧 TB6612#2 编码器排针要求外部逻辑电源，可接 `3V3`；若驱动板已给编码器供电，本列留空 |
| `C24` | `GND` | 右侧 TB6612#2 的 `RR` 编码器接口 `GND`，跳到 GND 轨 |
| `C25` | `RR_A` | M0 `PA26`、右侧 TB6612#2 的 `RR` 编码器输出 `A` |
| `C26` | `RR_B` | M0 `PA25`、右侧 TB6612#2 的 `RR` 编码器输出 `B` |

注意：编码器 `A/B` 高电平必须实测。若驱动板输出到 M0 的 `A/B` 是 5V，`LR_A/LR_B/RR_A/RR_B` 先走 `SN74LVC245A`，再进 M0。不要把 M0 `3V3` 和驱动板自带的编码器电源输出硬并联。

前轮 `LF/RF` 编码器第一版不接 M0；它们可以仍然插在驱动板的电机/编码器接口上。

### 3.4 IMU 与 OLED/I2C 区 C29-C38

| 列 | 网络 | 接法 |
| ---: | --- | --- |
| `C29` | `3V3` | IMU `VCC` |
| `C30` | `GND` | IMU `GND` |
| `C31` | `IMU_TX` | M0 `PA10 TX` -> IMU `RX` |
| `C32` | `IMU_RX` | M0 `PA11 RX` <- IMU `TX` |
| `C35` | `3V3` | OLED `VCC`，后续 PCA9685 `VCC` 也可插此列 |
| `C36` | `GND` | OLED/PCA9685 `GND` |
| `C37` | `SDA` | M0 `PA28`、OLED `SDA`、后续 PCA9685 `SDA` |
| `C38` | `SCL` | M0 `PA31`、OLED `SCL`、后续 PCA9685 `SCL` |

OLED 直接上车：

```text
OLED VCC -> C35 / 3V3
OLED GND -> C36 / GND
OLED SDA -> C37 / SDA
OLED SCL -> C38 / SCL
```

PCA9685 只接逻辑时：

```text
PCA9685 VCC -> C35 / 3V3
PCA9685 GND -> C36 / GND
PCA9685 SDA -> C37 / SDA
PCA9685 SCL -> C38 / SCL
PCA9685 V+  -> V5S 外部星形点，不走面包板
```

### 3.5 K230 UART / 激光控制预留区 C41-C46

| 列 | 网络 | 接法 |
| ---: | --- | --- |
| `C41` | `GND` | K230 `GND` 预留，也要回 `GND_PWR` |
| `C42` | `K230_TX` | M0 `PA17 TX` -> K230 `RX` |
| `C43` | `K230_RX` | M0 `PA18 RX` <- K230 `TX` |
| `C46` | `LASER_SW` | M0 `PB15`，只放控制线 |

K230 供电：

```text
K230 5V -> V5L 星形点单独拉线
K230 GND -> GND_PWR
不要从面包板下红轨给 K230 供主电
```

激光供电：

```text
KY-008 VCC -> V5S 星形点
KY-008 GND -> AO3400 漏极
AO3400 源极 -> GND_PWR
LASER_SW / PB15 -> 100R -> AO3400 栅极
AO3400 栅极 -> 100k -> GND_PWR
```

### 3.6 声光与按键区 C50-C63

只有一个三脚蜂鸣器模块。总图里多个 `B` 只是把 `C54-C56` 这一片标成蜂鸣器插线区域：`C54` 接 `VCC`，`C55` 接 `GND`，`C56` 接 `IO`。

| 列 | 网络/器件 | 接法 |
| ---: | --- | --- |
| `C50` | `START_KEY` | M0 `PB2`，接按键一端 |
| `C51` | `GND` | 按键另一端，跳到 GND 轨 |
| `C54` | `3V3` | MH-FMD 蜂鸣器 `VCC` |
| `C55` | `GND` | MH-FMD 蜂鸣器 `GND` |
| `C56` | `BEEP` | M0 `PB13`，MH-FMD 蜂鸣器 `IO` |
| `C58` | `LED_R` | M0 `PA9`，接 `R1 330R` 一端 |
| `C59` | `N_LEDR` | `R1` 另一端、红 LED 阳极 |
| `C60` | `GND` | 红 LED 阴极，跳到 GND |
| `C61` | `LED_G` | M0 `PA15`，接 `R2 330R` 一端 |
| `C62` | `N_LEDG` | `R2` 另一端、绿 LED 阳极 |
| `C63` | `GND` | 绿 LED 阴极，跳到 GND |

按键：

```text
PB2 / START_KEY -> C50 -> 按键 -> C51 -> GND
程序启用 PB2 内部上拉
```

LED：

```text
红灯: PA9  -> C58 -> R1 330R -> C59 -> 红 LED -> C60 -> GND
绿灯: PA15 -> C61 -> R2 330R -> C62 -> 绿 LED -> C63 -> GND
```

## 5. 上电前检查

万用表断电测：

```text
3V3-GND 不短路
V5L-GND 不短路
VBAT-GND 不短路
V5L 与 3V3 不直接导通
V5L 与 V5S 不导通
```

上电顺序：

```text
1. 只测两块 TB6612 板载 5V 输出。
2. 接 V5L -> 5V->3.3V 小稳压模块，先测 OUT 为 3.30V 左右，再接 M0 3V3。
3. 接面包板 3V3/GND/V5L，复测轨道电压。
4. 接 OLED、按键、LED、蜂鸣器，跑 IO 测试。
5. 接灰度、IMU、从 TB6612 板引出的编码器 A/B，跑传感器测试。
6. 接 TB6612 控制线，先不接电机，确认 PWM/DIR/STBY。
7. 接单电机悬空低 PWM 点动，再接四电机。
```
