# 北邮 2026 电赛小车任务程序说明

本工程入口为 `src/main.c`，目标板为 LP-MSPM0G3507。程序按当前 V3 引脚方案组织：底盘循迹、编码器、ATK-IMU901、OLED、PCA9685 云台/机械臂、K230 视觉串口、激光/执行器控制都已接入。

## 单按键任务选择

按住按键时小车保持静止，OLED 会显示当前选择档位；松开按键后启动对应任务。任务运行中再次按下并松开按键，会立即停车退出。

按住时长按 `700 ms` 一档计算：

| 档位 | 大致按住时间 | 功能 |
| --- | --- | --- |
| 1 | 0.0-0.7 s | 自动巡迹：A 点出发，一圈回 A 点停车并声光提示 |
| 2 | 0.7-1.4 s | 定点瞄准：停车状态下云台/机械臂对靶心完成一次动作 |
| 3 | 1.4-2.1 s | 巡迹到靶位联动：A->B 停车瞄准，再经 C、D、A，每个关键点声光提示 |
| 4 | 2.1-2.8 s | 动态瞄准：边循迹边持续跟踪目标 |
| 5 | 2.8-3.5 s | 同步绘制：停车，云台按预设轨迹绘制简单几何图形 |
| 6 | 3.5-4.2 s | 四圈连续运行：循迹 4 圈，并在行驶中完成一次完整瞄准 |
| 7 | 4.2-4.9 s | 工程训练/模块自检：灰度、IMU、左右编码器和电机方向自检 |
| 8 | 4.9-5.6 s | 传感器看板：停车显示灰度、IMU、里程、K230 状态 |
| 9 | 5.6 s 以上 | 直线测试：用编码器和 IMU 做短距离直行闭环 |

## OLED 显示

OLED 用于显示任务号、当前状态、灰度位置、灰度掩码、IMU 航向角和编码器估算里程。按住按键选择任务时，OLED 显示档位和任务名称。

## K230 串口格式

M0 侧按以下文本帧解析 K230 目标：

```text
$X:+012,Y:-008,Q:085#
```

`X/Y` 是靶心误差，`Q` 是识别质量。程序默认 `Q >= 30` 才认为目标有效。M0 引脚为 `PA17 TX -> K230 RX`，`PA18 RX <- K230 TX`，波特率 `115200`。

## 需要实车标定的参数

这些参数在 `src/board_pins.h` 中：

| 参数 | 作用 |
| --- | --- |
| `BOARD_LAP_DISTANCE_MM` | 一圈里程估计，默认 3200 mm |
| `BOARD_KEYPOINT_DISTANCE_MM` | B、C、D、A 的累计里程点 |
| `BOARD_LINE_KP` / `BOARD_LINE_KD` | 灰度循迹 PD 参数 |
| `BOARD_MOTOR_START_PERCENT` | 循迹基础速度 |
| `BOARD_SERVO_CENTER_DEG` | 云台中心角 |
| `BOARD_AIM_LOCK_ERROR` | 判定瞄准锁定的误差阈值 |

首次上车建议先跑第 7 档自检，再跑第 8 档看灰度/IMU/OLED，最后低速跑第 1 档自动巡迹。
